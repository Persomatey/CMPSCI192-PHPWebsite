Music-Chan

A music focused Discord bot code repository that, when linked with a Discord Bot using Discord's Developer Portal, will play a given music stream on runtime in a given Discord server's voice channel. Designed to be easy to impliment for any Discord moderator without requiring any prior coding knowledge. A full breakdown of the instructions on how to get the code running for your Discord bot can be found in the belowmentioned repository.

I solo developed this project, doing the programming in Javascript using the Node.js and Discord.js frameworks alongside FFmpeg and OpusScript libraries.
